# 数字电路基础实验
数字电路是处理器设计的期前导课程，在这一节利用 HDL 来描述电路模块，重点关注

- 信息的二进制编码
- 组合逻辑设计: 多路选择器, 译码器, 优先编码器, 加法器, 比较器
- 时序逻辑设计: 时钟, D 触发器, 计数器, SRAM 和 DRAM, 有限状态机, 时序分析

**要注意培养硬件思维，要先有电路再利用硬件描述语言来描述**
