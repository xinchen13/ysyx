## Tips
#### set environment variables
```sh
source tenv
```

#### clone this repository & update all the branches
```sh
git clone -o xinchen git@github.com:xinchen13/ysyx.git
cd ysyx
./github-up.sh
```

#### synchronize all the modifications to GitHub
```sh
./github-commit.sh "commit_message"
```
